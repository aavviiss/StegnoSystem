# Python StegnoSystem

> Python Flask Feedback app that sends data to Postgres database and emails user

## Quick Start

```bash
# Add your DATABASE URI in app.py and your mail params in send_mail.py

# Install dependencies
pipenv shell
pipenv install

pip3 install virtualenv
virtualenv env
pip3 install flask flask-sqlalchemy
python3 app.py 
pip install piping
pipenv shell
pipenv install flask
sudo pipenv install psycopg2
sudo pipenv install psycopg2-binary
sudo pipenv install flask-sqlalchemy
pipenv install gunicorn
sudo sudo pip install pil
python -m pip install pillow
python3 -m pip install Pillow
pip install steganography
pipenv install Stegano
pip install pysstv

# Serve on localhost:5000
python app.py
```
